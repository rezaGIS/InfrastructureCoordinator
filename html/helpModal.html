Colorado Resiliency Working Group

Project Tool Help

This tool should be used by project managers to display infrastructure projects with other groups in the State of Colorado.

  1. Create a project first begin by filling out the form.
  2. The only required fields are the project Start and project End dates, if you do not know the exact dates use a best estimate.
  3. Once form is complete, click "Save".
    • The form will now be locked and unable to be altered. To make changes hit "Cancel".
  4. After saving the form, begin creating the spatial aspect of the project, click “Draw” button to enable editing.
    • Click on map to create a vertices. Shape your project area and double click to end polygon creation.
    • If a mistake is made, click "Cancel" to clear the graphic drawn.
  5. When project creation is complete submit the project by clicking "Submit Project".
    • You will be prompted whether you confirm submission, click "Ok".
    • Click the "Show" button if the projects are not visible.
